2: Identical Diamonds

The key word to watch out for is "identical" diamonds. This means that it doesn't matter what diamonds someone has, but only what number of diamonds someone has.

3: It looks the same!

The order that you give the diamonds to the people doesn't matter. Which diamonds you give people doesn't matter either!

4: Let's explore one interesting counting method.

There are other ways to count the number of ways to distribute 5 identical diamonds among 3 robbers. This is just one way.

5: Let's start by rearranging the diamonds and people boxes.

We can rearrange the lines and diamonds so they look more like this.

6: They are all arranged in a line!

Notice how the order of the diamonds and lines shows how many diamonds a robber has.

7: The distribution of diamonds has changed.

One diamond was taken from robber 1 and given to robber 2.

8: There are too many lines.

The diamond has to cross over two lines in order to be considered as belonging to another person. If the diamond only crosses over one line, the diamond does not belong to any robbers. We don't want that, as we want to leave the museum with all of the diamonds.

9: We can get rid of unnecessary lines.

We can combine the two lines in the middle into one line. And remove the lines on the edges as they also don't mean anything.

10: Changing the position of these lines changes how many diamonds each robber has.

To find out the total number of ways we can distribute the 5 diamonds to the 3 robbers, we just need to find the total number of ways we can order the lines and diamonds.

11: Take this time to try to think about how we can do this.

Try it out! Grab a pencil and a sheet of paper and try to figure it out!

12: Remember that all of the diamonds are the same.

We already know that since all of the diamonds are the same changing the order of these diamonds doesn't matter.

13: Imagine there are slots.

We can create slots in between the diamonds where we can place our two lines.

14: Lets place the lines in different slots.

See how placing the lines in different slots changes how many diamonds a robber has?

15: So there are four slots.

Let's number them!

16: Imagine that we can reorder these slots and the first two will be where we place the lines.

For example, if the order is 4, 3, 2, 1. Then the fourth slot and the third slot is where we would place the lines.

17: There are four options for the first slot, three options for the second slot, two options for the next slot, and one option for the last slot.

So we could find the total number of combinations by multiplying those numbers: 4 x 3 x 2 x 1.

18: We don't necessarily care about the last two since they are blanks.

For the last two, there are 2 options for the first slot, and 1 option for the last slot. So, 2 x 1 total number of combinations.

19: We can divide the total number of combinations for yes and no, by the total number of combinations for no.

4 x 3 x 2 x 1 / 2 x 1.

20: We also don't necessarily care about the order of the first two either.

They are both identical lines and the order of them doesn't matter.

21: In the same way, there are two options for the first one and two options for the second one.

The total number of combinations is 2 x 1.

22: Therefore we can divide the final number again by (2 x 1).

(4 x 3 x 2 x 1/ 2 x 1)/(2 x 1)
